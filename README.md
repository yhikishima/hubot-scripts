# hubot-yh

yh

See [`src/yh.coffee`](src/yh.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-yh --save`

Then add **hubot-yh** to your `external-scripts.json`:

```json
[
  "hubot-yh"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
